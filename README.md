
## KraudDonate-Auth

KraudDonate-Service is part of the KraudDonate application to help volunteers.
It is used to develop authorization.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```


## Technology

    * NestJS
    * TypeScript
    * PostgreSQL
    * Prisma ORM
